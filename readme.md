# DraftMVC Boilerplate

## Get started

### Create 'config/db.json'

```json
{
    "host": "localhost",
    "db": "draftmvc_boilerplate",
    "user": "root",
    "password": "root"
}
```

### Install gulp (for building scss / js)
```bash
npm install
```

### Install composer packages
```bash
composer install
```

### Run gulp
```
npm run gulp
```
